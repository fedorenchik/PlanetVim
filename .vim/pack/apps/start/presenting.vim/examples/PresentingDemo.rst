           **presenting.vim**

          a presentation tool

                for vim

~~~~

========================================
Usage
========================================

Slides are separated by four ~

Start presenting::

  :PresentingStart

Navigation:

 * n - next slide
 * p - previous slide
 * q - quit

~~~~

========================================
Inline Code
========================================

RST automatically supports syntax hl
for python code!

.. code:: python

    if __name__ == '__main__':
        import sys
        sys.exit()
